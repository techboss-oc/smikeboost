# SmikeBoost On-Page SEO & Copy Playbook (StoryBrand Driven)

> Goal: Give every public page a clear narrative arc (hero = Nigerian creator/agency, guide = SmikeBoost, plan = simple steps, success = social proof, failure avoidance = stalled growth). Copy below is ready-to-use text you can paste into the respective sections.

---

## Global Guidelines

- **Voice:** Confident mentor from Nigeria’s creator economy; mix professional clarity with energetic phrases common to Lagos tech/startup circles.
- **Primary CTA:** `Create a free SmikeBoost account`
- **Secondary CTA:** `Talk to a Lagos strategist`
- **Keyword Spine:** `SMM panel Nigeria`, `buy Instagram followers in Lagos`, `NGN social media marketing`, `TikTok boost Nigeria`, `Naira payments SmikeBoost`, `Flutterwave SMM deposit`, `reseller API Nigeria`.
- **Internal Links:** Every page should link to `services`, `pricing`, `faq`, and at least one blog resource matching the topic cluster.
- **Schema:** `BreadcrumbList`, `FAQPage`, `Service`, `HowTo`, `Product`, and `SoftwareApplication` where relevant. Keep OG locale `en_NG`.
- **Images:** Compressed WebP hero (<80 KB) + descriptive alt tags emphasising Nigeria context.

## Copywriting Frameworks Applied

- **StoryBrand**: Character → Problem → Guide → Plan → Success → Failure Avoidance.
- **PAS (Problem/Agitate/Solve)** inside each section.
- **Local Proof**: Mention specific Nigerian cities, banks, and payment rails.
- **Trust Triggers**: NDPR compliance, uptime transparency, real numbers.
- **Action Language**: Imperative verbs, no passive voice, short sentences (<16 words) for readability.

## Deployment Checklist

1. Update each PHP view with the recommended hero/subhead/section copy.
2. Refresh meta tags by calling `get_seo_tags` with new titles/descriptions.
3. Add structured data (FAQ, HowTo, Service) via JSON-LD per page.
4. Compress hero imagery & update `asset()` references with descriptive alt text.
5. Run Lighthouse + Screaming Frog to confirm 10/10 on-page scores.
6. Submit updated URLs to Google Search Console for recrawl.
